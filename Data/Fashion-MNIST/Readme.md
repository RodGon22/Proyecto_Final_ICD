# Conjunto de Datos Fashion-MNIST

**Autores**: Han Xiao, Kashif Rasul, Roland Vollgraf  
**Fuente**: Zalando Research  

**Cita**:  
Han Xiao and Kashif Rasul and Roland Vollgraf, *Fashion-MNIST: a Novel Image Dataset for Benchmarking Machine Learning Algorithms*, arXiv, cs.LG/1708.07747

## Descripción

Fashion-MNIST es un conjunto de datos de imágenes de artículos de Zalando que consiste en:
- **Conjunto de entrenamiento**: 60,000 ejemplos
- **Conjunto de prueba**: 10,000 ejemplos

Cada ejemplo es una imagen en escala de grises de 28×28 píxeles, asociada con una etiqueta de 10 clases. Fashion-MNIST está diseñado para servir como reemplazo directo del conjunto de datos MNIST original para la evaluación de algoritmos de aprendizaje automático. Comparte el mismo tamaño de imagen y estructura de divisiones de entrenamiento y prueba.

**Datos originales disponibles en**: https://github.com/zalandoresearch/fashion-mnist
